Instruction/Data operands
Flags structure

Detailed Description

Represent instruction/data operands.

IDA keeps 2 bitmasks:

  • representation of the first operand
  • representation of other operands (we will call this 'representation of second operand' although it is also applied to third, fourth, etc operands too)

For data bytes, only the first bitmask is used (i.e. all elements of an array have the same type).
